# Othello

This is a classic othello game.

## How to start

### SSH

```
git@github.com:PrescilliaDSD/othello-prescilliadsd.git
cd othello-prescilliadsd
npm install
npm start
```

### HTTPS

```
https://github.com/PrescilliaDSD/othello-prescilliadsd.git
cd othello-prescilliadsd
npm install
npm start
```
